# Estrutura do site

## 1. Hero / primeira dobra

Objetivo:
- gerar impacto imediato;
- transmitir sofisticação;
- parecer premium;
- mostrar qualidade visual logo nos primeiros segundos.

Usar:
- `hero-01.jpg`

Elementos:
- título forte;
- subtítulo curto;
- botão de CTA;
- overlay escuro;
- animações suaves;
- aparência cinematográfica.

Evitar excesso de texto.

Objetivo:
