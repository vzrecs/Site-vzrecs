# Tipografia

## Fonte para títulos

Fonte principal para títulos:
Neue Haas Unica W1G

Arquivo:
04-identidade-visual/fontes/NeueHaasUnicaW1G-Bold.woff2

Uso:
Títulos principais, chamadas de impacto, headings de seção, frases fortes e palavras em destaque.

Estilo:
Usar os títulos em caixa alta, com aparência premium, limpa, moderna e sofisticada.

Orientação de espaçamento:
Usar espaçamento entre letras um pouco mais próximo do que o padrão original, para criar uma sensação mais premium e compacta.

Exemplo técnico para o Codex:
Aplicar letter-spacing levemente negativo nos títulos, entre -0.02em e -0.05em, ajustando visualmente para não prejudicar a leitura.

Também usar line-height mais compacto nos títulos, entre 0.9 e 1.05, dependendo do tamanho.

## Fonte para textos

Fonte principal para textos:
Inter bold italic 

Uso:
Parágrafos, descrições, menu, botões, textos secundários, informações de apoio e navegação.

Estilo:
Limpa, legível, moderna e profissional.

Orientação de espaçamento:
Aplicar letter-spacing levemente negativo, entre -0.02em e -0.05em, ajustando visualmente para não prejudicar a leitura.

## Botões

Usar Inter com peso maior, Bold.

Os botões devem ter aparência moderna, limpa e premium.

## Hierarquia visual

Títulos:
Neue Haas Unica W1G, caixa alta, peso forte, espaçamento levemente negativo e impacto visual.

Textos:
Inter, leitura confortável, cor branca ou cinza claro, sem excesso de peso.

## Orientação para o Codex

Usar Neue Haas Unica W1G para todos os títulos e Inter para todos os textos.

Não usar mais de duas famílias principais de fonte.

Aplicar títulos em maiúsculas.

Ajustar o espaçamento dos títulos para ficar mais próximo, com sensação premium.

Não apertar demais textos longos.

Manter o site clean, escuro, premium, moderno e cinematográfico.
